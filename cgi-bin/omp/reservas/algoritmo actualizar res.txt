Algoritmo actualizacion de Reser
--------------------------------

Para cada registro bibliografico	
	Para cada parte	
		para cada inventario de la parte (hnos)		
			buscar inventario en reservas
			Si el inv existe
				actualizar estado
			sino
				regReserv = no existe
				repetir hasta encontrar registro
					buscar x en Reser
					si existe
						regReserva = registro de res de x
					fin si
				fin repetir
				Si regReserva = no existe
					agregar un nuevo registro con Inv y todos sus Hnos
				sino
					agregr inv a regReserva
				fin si
			fin si
		fin para
	fin para
