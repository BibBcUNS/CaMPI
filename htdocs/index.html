El sistema incluye varios módulos que le permitirán llevar a cabo tareas de gestión bibliotecaria. Hemos incluido en el demo algunas bases de datos de prueba mediante las cuales podrá conocer el funcionamiento del producto.

  • El módulo de circulación permite realizar:

    • Consultas por apellido o nro. de documento del usuario, nro. de inventario del material a prestar.
    • Devolución rápida por nro. de inventario.
    • Préstamo, reserva y renovación de libros.
    • Acceso a los datos del usuario y suspensión a morosos.

  • Mediante el módulo de administración es posible:

    • Definir el calendario.
    • Gestionar políticas de circulación.
    • Administrar datos de usuarios (lectores).
    • Generar listados (morosos, préstamos del día, devoluciones, estados de circulación).
    • Generar credenciales y códigos de barras para los materiales.
    • Actualizar inventarios cuando se creen nuevos registros en la base bibliográfica.

  • En el módulo de catalogación se podrá acceder a Catalis bibliográfico para crear registros MARC21/AACR2, y a Catalis autoridades, como así también a otros sitios de interés relacionados específicamente con el módulo de catalogación.

  • El módulo de estadísticas permite generar historiales por rango de fecha y turno de los préstamos, devoluciones, morosos, sanciones.

  • En el módulo OPAC se puede consultar el catálogo en línea.
